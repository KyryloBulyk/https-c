# 🌐 HTTPS in C

Welcome to the **HTTPS in C** project! 🚀 This project is a learning journey into the world of networking, encryption, and secure communication using the C programming language. If you're curious about how HTTPS servers work at a low level, you're in the right place! 🎯

## 📚 Project Overview

This project aims to build a basic HTTPS server from scratch using TCP sockets and OpenSSL for TLS/SSL encryption. We're diving deep into the details of how secure connections are established and how data is safely transmitted over the internet.
